# Project Plan: Encoded Brand Enterprise Deck
# Path: projects/encoded-brand-deck/README.md

## The Objective
Build a 12-slide enterprise sales deck using the "Page 7 Embed" pattern popularized by Nate's Newsletter (2026-04-24). The deck sells Bruiser's **Brand.md** and **Brand MCP** protocol to enterprise buyers.

## The Structure (12 Slides)
1. **Title**: The Disintegration of the Agentic Brand.
2. **Context**: You have 100+ agents representing you. None of them have been briefed.
3. **The Gap**: Brand Guidelines (PDFs) are for humans. Machines are guessing.
4. **The Solution**: Brand.md — The machine-readable system prompt for your org.
5. **The Protocol**: Brand MCP — Real-time connectivity for every agent in your stack.
6. **The Architecture**: Internal Coherence vs. External Representation.
7. **PROVE IT**: The Side-by-Side Embed. (Generic LLM vs. Brand.md-Grounded LLM).
8. **Scale**: How one file governs 1,000,000 interactions without manual review.
9. **Governance**: Version control for brand identity.
10. **Economics**: Replacing the "Mockup-to-Production" cost with direct intent.
11. **Consultancy**: Why the protocol is the floor and strategy is the ceiling.
12. **Close**: Bruiser — We wrote the playbook.

## Technical Stack
- **Medium**: HTML/React (deployed via Cloudflare Pages for live embedding).
- **Styling**: Tailwind CSS + Brand.md tokens.
- **Interactivity**: Page 7 features two chat windows connecting to an LLM.
- **Deployment**: `https://encoded-brand-deck.bruiser.co` (or similar).
